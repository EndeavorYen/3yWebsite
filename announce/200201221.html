[公告] 新增 HELP ROBOT 和 HELP MULTI

HELP ROBOT
機器人( ROBOT )之認定原則

發呆機器人( ROBOT ), 需符合以下三項原則
1.不論何處
2.不攻擊, 不施法
3.一至二分鐘之內, 做一次動作(或採取不影響他人之其他行為)
符合以上三項原則, 視為合法之發呆 ROBOT
確認為合法發呆 ROBOT 者, 擁有不被系統質詢問題之豁免權。

除發呆機器人外, 其餘各種機器人在本 mud裡, 皆視為違法
違法機器人, 一經檢舉, 確實查證有非法事宜者, 將採取系統質詢問
題之方式予以確認是否為機器人, 請各玩家密切遵守。

關於合法發呆機器人認定原則第二項之不攻擊, 不施法部份,
若屬自動攻擊之怪物所行使之攻擊行為者, 視以下部份認定:
1.若為國家怪物, 認定為非法之機器人
(請各國家注意)
2.若為系統既定之自動攻擊怪物者, 認定為非法之機器人
(請各發呆機器人, 確定所處地方是否有自動攻擊怪物)
3.若為系統既定會亂跑亂飛的自動攻擊怪物者, 認定為合法
(此因, 非屬站著發呆者之過)
4.若為他人任意施展魔界之門召喚之女鬼者, 視其地點為何
A.國家, 認定為違法
(此因, 國家應屬同志範圍, 不應有互鬥互毆之相關行為)
B.玩家房間, 認定為違法
(此因, 玩家房間應屬個人私人房間, 其內部之行為應不包含攻
擊事件)
C.系統區域, 認定為合法
(此因, 系統區域為公開公正之區域, 如選擇系統區域發呆者,
應接受在系統區域裡所發生之任何行為)
5.若為玩家攻擊玩家者, 視其地點為何
A.國家, 認定為違法
(理由同 4.A )
B.玩家房間, 認定為違法
(理由同 4.B )
C.系統區域, 認定為合法
(理由同 4.C )

HELP MULTI
有關多重連線( MULTI )之認定原則

多重連線( multi )意指：同一個人操縱多個玩家角色(char)，即認
定為多重連線( multi )。

有關於多重連線部份，其認定原則如下：
1.一般 multi( 非屬工作站 ), 皆由系統自動捉取。
此種 multi, 因其證據確鑿, 認定為罪有應得, 不接受任何反駁

2.ADSL multi, 因無法確定是否為多台電腦多人使用, 所以此部份
之 multi, 一律認定為確實 multi, 請使用 ADSL 者自行注意。

3.工作站 multi, 由於舉證困難, 將採取警告方式, 一旦被警告者
, 若仍無改善, 將關閉該工作站連線權利, 此部份之認定, 概由
神族自行判斷, 如有任何不服之處, 將由該神族提出判斷之緣由
。請使用工作站者注意, 若因個人操守, 導致工作站被關閉連線
權利, 請各工作站使用者自行負責。
