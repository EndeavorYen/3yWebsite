新手教戰守則

屬性篇:
最近有很多的新手來到這裡,問的問題千篇一律,所以整理一下僅供參考.
一開始玩家可以從hp sc gr report tra 等指令了解自己的狀態及屬性.
當升級時會依玩家的屬性及部份亂數(大概20%) 增加hp mp mv 以及訓練點數.
而訓練點數可以用來找 訓練師 提升屬性 而要增加哪一種 則由玩家決定.
而當玩家要領悟或學 新的skill時 有些skill 也會限制 玩家所必備的點數,
不然會領悟不出來或無法學習喔

train /? ---> 關於屬性的基本介紹
help attribution---> 屬性的影響及功能
train !list ---> 目前區域有哪些mob能幫你訓練

最後
每種職業都有其屬性限制如下:

刺客: str40 int30 wis30 dex40 con40
郎中: str30 int40 wis50 dex30 con30
鑄劍: str30 int45 wis40 dex35 con30
伶人: str30 int40 wis40 dex40 con30
天師: str30 int50 wis40 dex30 con30
盜賊: imm 幫我補 我沒資料

技能致能篇:
一開始的lv1~lv15 應該是適應期
只要待在新手區 陣亡的機會應該不大
不過有一種可能 就是沒mv 又被圍爐
在這裡當你k mob時 若有等級相進的mob 站同一格 就有可能會自攻擊 形成圍爐
此時若沒mv 掛掉的機會 相當大 就算hero 也一樣
若發現沒mv了 請猛按flee <---逃跑技能 也許能逃過一劫 省下一張金牌
所謂exp誠可貴 金牌價更高ㄚ 聽我的準沒錯 ^O^
所以 mv 相當重要 若不想常常按 report 或 gr 就要設 prompt <--相關設定請看 prompt /?

致能:
剛開始進遊戲時 會有基本技能可選 分為 攻擊 防禦 逃跑 三種.
打 le 會有你所有會的技能列表 打○的代表致能 打Ｘ的反之
而致能代表 你已經將此技能 掛上 會自動使用
像法術無法致能 就必須 用 case 'skillname' mob
如何致能ㄋ ena 'skillname' 使用第二次 就會將致能取消
看mob 會哪些技能 用 le mob(name) !list 向mob 學技能 用 le mob(name) 'skillname'
leftskill 可以知道自己現在能學的技能 配合pry 可以知道 去哪學 或由哪種技能領悟
想知道技能的相關資訊 用 pry 'skillname' 不過一次要兩千
當你pry 後會有以下的資訊 以雙十旋龍斬為例

英文名稱﹕sky dragon sword
中文名稱﹕雙十旋龍斬
攻擊武器﹕長劍
互相教導﹕不可以
技能功能﹕攻擊類
技能分類﹕技能
浪費數值﹕體力
領悟技能﹕從缺
預備功夫﹕神龍劍法(fu sword)
領悟機率﹕很低
職業限制﹕ 將 軍則等級最低要 一級才能學習﹐熟練度最高可以到登峰造極﹗
　　　　　 刺 客則等級最低要 一級才能學習﹐熟練度最高可以到一代宗師﹗
限 制﹕一、本技能限制體格不能低於二十五。
　　　 二、本技能限制敏捷不能低於二十五。
　　　 三、本技能限制力量不能低於二十五。
　　　 四、本技能限制技能神龍劍法熟練度不能低於為神乎其技。
　　　 五、本技能限制職業限制為 將軍 刺客

當你想領悟時 請看完 以上限制 才努力的練 不然會白費力氣
戰鬥前 請先lore mob(name) 這樣會告訴你 k他有沒有危險 基本上k 上下三級 exp最多
上下六級 有exp 還有k mob時 儘量用全名
像 k dog 最好不要只打 k d 如果剛好 有一個叫 dXXX 的玩家 走過來 那就......恭喜你了
設定 config config的設定是打 config (+/-)選項 打圈的表示 + 有使用 打差反之

交通區域 recall 設定篇:
交通:
一開始玩家都在出生地活動，每個出生地都有必備的設施、人物，所以不假外求，但長大了後 會想到外地去升級，就會需要交通工具，三國的交通工具有地鐵、飛機和船，地鐵請打bus 可以知道列車經過的城市和入站時間，船和飛機打ship，同上，要上船請打 ship on，地鐵則不用，會自動上車

recall:
當你到其他城市時 會看到在該房間名字前有 * 的符號
表示該格能設recall點 共有1~10 十個點可記憶 但並非每個城都有喔!!
設定方法 recall set 1~10 設完用 recall 1~10 就可以 用飛的 瞬間到達該城市了
方便 簡單 好用 喔~~ 要謝謝imm 區域

area:
當你打 area 時會出現一堆區域
編號 編輯者 區域名稱 房間 怪物 物品 商店 重置 玩家 等級

要看該區的敘述 請用area help (編號) 打--> area help 33 顯示

裝備篇:
裝備不只幫你抵消mob的攻擊 還會加屬性 和 hp mv mp喔 不過他會壞掉..... 打eq 可以看到你身上裝備著 哪些eq喔 設定裝備的防禦 % 數 eq !set (10~90) 設的越大越容易壞喔 但你比較不會受傷 修eq : 壞了就要修嘛 去找鐵匠 打 repair (eqname) (錢) <---不用()喔 還有eq所加的點數 在skill的限制上是可行的喔 如果 skill 要con 20 但你只有con 18 的話 快去找+con的eq喔 要善用喔 粉重要喔

你要買東西 打 bet (出價)--->最少為目前價錢+20%
別人拍賣時 打 auc 可以看到物品的等級喔 <---很有用
