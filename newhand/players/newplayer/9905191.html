alias很好用

我把一些經驗告訴大家
一些常常做的動作 可以用alias

例如修eq
修eq很麻煩 就可以用alias

設了alias rr remove %1:repair %1 %2:wear %1
　　　　　|　 |　　 |　　
　　　　　|　 |　　第三個指令　
　　　 　 |　第二個指令
　　　　 第一個指令

你所想要的簡寫 當我要補皮帽(hat) 找到鐵匠(smith)<---崇拜....
他什麼都會修
在smith面前鍵入 rr hat 100
　　　　　　　　 | | 　|
　　　　　　　　 | | 代表%2(第二個變數)
　　　　　　　　 | 代表%1
　　　　　　　　就是簡寫

那結果就會變成
remove
你脫下滿是破洞的帽子(hat)
repair hat 100
你蠻不好意思的將皮帽連同100元遞給鐵匠小姐
鐵匠小姐抿著笑說:[怎麼穿成這樣??]
妳看著她將100元糊在帽上遞還給你
修好啦!!

wear hat 你很高興的戴上帽子
感動的對她說:[謝謝妳...]
好了...跟實際情形或許有出入...但是差不多啦
那你不久就會發現 如果我要修第二個戒子怎麼辦??
這也很簡單 可以改成alias rr2 rem 2.%1:repair %1 %2:wear %1 就可以了
